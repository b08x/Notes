====== openMHA ======
Created Monday 02 August 2021


to start interactively;
change into the directory with the config you want to run

cd ~/Workspace/openMHA/configs
mha --interactive

then to read in a file

?read:adm.cfg
cmd=start

then to stop
cmd=stop
